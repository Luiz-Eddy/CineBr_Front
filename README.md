# CineBr Front End Code
Desenvolvimento Front End do CineBr. Projeto para faculdade UniBh

O projeto possui o objetivo de desenvolver uma plataforma de straming de conteúdos produzidos no mercado nacional. Dando maior visibilidade para produções do nosso país e produções independentes ou pequenas.

Esse repositório é exclusivamente para armazenar o código base do desenvolvimento front-end deste projeto.

O uso é livre.
